isTemplate

Checks is the file has a template file extension.

isTemplate(file: String, eFiles: Boolean): (String | null)
Parameters
file (String) Filename
eFiles (Boolean) true : All e files like .exml, .emap, etc. are possible; false : Only *.emap is possible.
Returns
(String | null): If the file is a template, it will return the file without the e in the file extension like filename.map (from filename.emap ).
Example
const isTemplate = require(`mapfile-ejs`).isTemplate;

console.log(isTemplate(`filename.emap`,false)); // filename.map
console.log(isTemplate(`filename.ecss`,false)); // null
console.log(isTemplate(`filename.ecss`,true));  // filename.css
console.log(isTemplate(`filename.map`,false));  // null
console.log(isTemplate(`filename.css`,false));  // null

render

Renders a ejs template file.

render(inputFile: String, outputFile: String, options: Object?): Boolean
Parameters
inputFile (String) Path of ejs template file
outputFile (String) Path of output file
options (Object?) Options of this function
Name Description
options.inputEncoding String (default `utf8`) Encoding of the input file (see Supported encodings ).
options.outputEncoding String (default `utf8`) Encoding of the output file (see Supported encodings ).
Returns
Boolean: true : Rendering was successful, false : Rendering wasnâ€™t successful
Example
const renderFile = require(`mapfile-ejs`).renderFile;

// render file with default options
render(
  `template__utf8.emap`,
  `template__utf8.map`
);

// render file with defined output encoding
render(
  `template__utf8.emap`,
  `template__iso_8859_1__01.map`,
  { outputEncoding: `ISO-8859-1` }
);

// render file with defined input and output encoding
render(
  `template__iso_8859_1.emap`,
  `template__iso_8859_1__02.map`,
  { inputEncoding: `ISO-8859-1`, outputEncoding: `ISO-8859-1` }
);

watch

Watches a directory and renders ejs templates on change.

watch(dir: String, options: Object): Object
Parameters
dir (String) Path of the directory
options (Object) Options of this function
Name Description
options.inputEncoding String (default `utf8`) Encoding of the input file (see Supported encodings ).
options.outputEncoding String (default `utf8`) Encoding of the output file (see Supported encodings ).
options.ignoreInitial Boolean (default false) true : Ignore rendering of files on initial, false : Render all file on inital
options.eFiles Boolean (default false) true : Render all e files like .exml, .ecss, false : Render only *.emap
Returns
Object: Watcher object, see https://github.com/paulmillr/chokidar
Example
const watchDir = require(`mapfile-ejs`).watch;

// watch a directory with default options
watch(`dir1`);

// watch a directory with custom options
watch(`dir2`, {
  inputEncoding: `utf8`,
  outputEncoding: `utf8`,
  eFiles: true,
  ignoreInitial: true
});
